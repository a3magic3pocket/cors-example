# cors-example
cors-example

# How to run a api server
- ```bash
    # Open new terminal 
    cd api
    go run main.go
    ```

# How to run a front server
- ```bash
    # Open new terminal
    cd fron
    go run main.go
    ```

# Check both servers are running
- Open a chrome browser
- Access 'http://localhost:8888'
- Click 'POST 요청' button 
- Press F12 and check console log
